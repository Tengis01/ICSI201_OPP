# ICSI201_OPP


За лаборатор 9
Шинэ баг болж хуваагдсан байгаа. Тэгэхээр одоо доорх байдлаар ажилаа хийх болно. Энэ удаа цоо шинээр класс диаграмын дагуу код бичихээр байгаа
Класс диаграммых нь зургийг харсан байх. 6 ширхэг класс байсан хуваалтыг нь доорх байдлаар хийлээ сонгоод авна шүү хараарай. Тэгэхээр нэг хүн 2 классын код бичээд нэмээд сонгосон pptx даалгаварыг хийнэ. 

Онол - Эрдэмбилэг + Employee + Child
Агуулга зорилго дүгнэлт -              +   People + Spouse + Тайлан засвар эмхэтгэл  
Хэрэгжүүлэлт, Үндсэн код -             +   Division + Job description

Код бичилт header үүсгээд бичих эсвэл ганц .cpp файл дотор бүгдийг нь бичихийг үндсэн код авах хүн шийдсэн нь дээр байхаа эсвэл ярилцаж болно шүү. 


Агуулга, Зорилго, Онол, Хэрэгжүүлэлт, Дүгнэлт, Ашигласан материал, Хавсралт энэ бол багшийн өгсөн агуулга энүүгээр нь хийссэн нь дээр байх.
Анхаарах зүйл нь pptx мэдээлэл хүнд уншиж танилцуулах болохоор түүнийгээ бодолцоод хийгээрэй.
