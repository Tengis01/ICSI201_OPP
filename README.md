За лаборатор 10
Лаб 9 дээрх үр дүнг харахад манайд сайжруулах болон давуу байсан зүйлсээ түрүүлээд бичье. Манай багийн илтгэх ур чадвар бол маш сайн байсан гэж би бодож байна. Бас слайд шилжүүлэхдээ их гарамгай хурц мэдрэмжтэй хүмүүс байналээ шүү. Сул зүйлс pptx дээрх текст болон дэд гарчигийн хэмжээ эзлэж байгаа зай байрлал зэрэг зохимжгүй байлаа. Мөн цаг давчуу байсан болохоор багийн гишүүд бусдыхаа хийсэн зүйлсийг уншиж ойлгож өөрийн болгож чадаагүйн хамгийн том сул тал байсан ч амжилттай давлаа баяр хүргэе. 


За энэ удаагийн лабораторын ажилаар Template гэх ойлголт орж ирж байсан. Хийх ажилын ерөнхий загвар хувиарилалт өөрчлөх бодолгүй байна. Хэдий заавар дээр онолын хэсэгийн даалгавар оруулаагүй байсан ч энэ бол зайлшгүй байх шаардлагатай. Тиймээс голдуу код дээр ажиллах ажил болох шинжтэй. Өөрийн ажлыг сонгохдоо өмнө аваагүй хэсгээ авна шүү. Энэ юу вэ гэвэл жишээ нь онол бичсэн хүн нөгөө 2оос сонголтоо хийнэ. Учир гэвэл олон төрлийн чадварыг сайжруулах тэнцвэртэй байлгах зэрэгт оршино.
